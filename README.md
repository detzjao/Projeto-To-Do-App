# 💻 Projeto To-Do App

Aplicação web simples para gerenciamento de tarefas (To-Do List), permitindo criar, visualizar, editar e remover tarefas de forma prática.

---

## 📌 Sobre o projeto

Este projeto foi desenvolvido com o objetivo de praticar conceitos fundamentais de desenvolvimento web, como:

* Manipulação do DOM
* Estruturação de código
* Lógica de programação
* Organização de projetos front-end

Além disso, o sistema simula um cenário real de uso, onde usuários precisam organizar suas atividades do dia a dia.

---

## 🚀 Funcionalidades

* ✅ Criar tarefas
* 📋 Listar tarefas
* ✏️ Editar tarefas
* ❌ Remover tarefas
* ✔️ Marcar como concluída

---

## 🛠️ Tecnologias utilizadas

* HTML5
* CSS3
* JavaScript

## ▶️ Como executar o projeto

1. Clone o repositório:

```bash
git clone https://github.com/detzjao/Projeto-To-Do-App.git
```

2. Acesse a pasta:

```bash
cd Projeto-To-Do-App
```

3. Abra o arquivo `index.html` no navegador
